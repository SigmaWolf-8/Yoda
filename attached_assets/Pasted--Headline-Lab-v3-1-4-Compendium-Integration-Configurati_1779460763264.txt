## 2. Reserved Code Points | Only Two Permanent Categories

The Unicode Standard permanently reserves code points for two purposes:

1. **Surrogates** (U+D800-U+DFFF, decimal 55296-57343): 2,048 values used only by UTF-16. They are **not** Unicode scalar values. Rust's `char` cannot hold them; a valid UTF-8 `String` will never contain their byte sequences.
2. **Noncharacters** (66 values): These *are* scalar values, so Rust's `char` can represent them and a `String` may contain them (intended for internal use). The 66 noncharacters consist of:
   - 32 contiguous points: U+FDD0-U+FDEF (decimal 64976-65007)
   - 34 points formed by the last two code points of each of the 17 planes: U+*xx*FFFE and U+*xx*FFFF.

**There is no third category.** Unassigned code points may be assigned in future Unicode versions. Private-use areas (U+E000-U+F8FF, U+F0000-U+FFFFD, U+100000-U+10FFFD) are normal scalar values. Rust treats them as ordinary `char`s. Thus only surrogates and noncharacters are permanently reserved, and Rust's handling of them is principled and transparent.

## 3. Unicode Code Points and the 3-D Space

A code point is an integer in the range `0 ..= 1_114_111` (U+0000 .. U+10FFFF). The entire space is a 3-D structure: **Plane** (17 planes, 0-16), **Row** (256 per plane, hex 00-FF), **Column** (256 per row, hex 00-FF). Total: 17 x 256 x 256 = 1,114,112.

Key hex-to-decimal: F = 15, FF = 255, FD = 253, D8 = 216, DF = 223, FE = 254.

### 3.1 Plane-First (Primary = Plane)

Iterate by plane, then row, then column. Standard Unicode numeric order. Surrogates (U+D800-U+DFFF; 55296-57343) are contiguous in Plane 0 rows D8-DF (216-223). Noncharacters Category A (U+FDD0-U+FDEF; 64976-65007) in Plane 0 row FD (253) columns D0-EF (208-239). Category B at end of each plane (U+xxFFFE, U+xxFFFF).

### 3.2 Row-First (Primary = Row)

Iterate by row across all planes before moving to next row. Surrogates only in Plane 0 rows D8-DF. Category A only in Plane 0 row FD. Category B in row FF across all planes at columns FE and FF | repeating pattern.

### 3.3 Column-First (Primary = Column)

Iterate by column across all rows and planes. Surrogates scattered. Category A spread across 32 columns. Category B tightly clustered at columns FE and FF | every 256th entry at row FF of each plane is a noncharacter.

### 3.4 Summary

None of the three orderings is "correct" | each serves a different analytical purpose.

## 4. UTF-8 Encoding | Bit-Level Walkthrough

- **1-byte** (U+0000..U+007F): `0xxxxxxx` (7 payload bits)
- **2-byte** (U+0080..U+07FF): `110xxxxx 10xxxxxx` (11 bits)
- **3-byte** (U+0800..U+FFFF): `1110xxxx 10xxxxxx 10xxxxxx` (16 bits)
- **4-byte** (U+10000..U+10FFFF): `11110xxx 10xxxxxx 10xxxxxx 10xxxxxx` (21 bits)

Rust's `str::chars()` decodes these bytes, producing a `char`. In our system, a `char` is immediately converted to a `TritSeq` via the gate.

## 5. `TritSeq` | Exact Variable-Width Ternary Representation

```rust
#[derive(Debug, Clone, Copy, PartialEq, Eq, PartialOrd, Ord, Hash)]
pub enum Trit { Zero, One, Two }

#[derive(Debug, Clone, PartialEq, Eq, Hash)]
pub struct TritSeq(Box<[Trit]>);

impl TritSeq {
    /// `from_char` is the gate: binary integers are used freely inside
    /// to extract the scalar from Rust's `char` primitive. Outside this
    /// gate, no `u32` participates in any computation or comparison.
    pub fn from_char(c: char) -> Self {
        let cp = c as u32;
        if cp == 0 {
            return TritSeq(Box::new([Trit::Zero]));
        }
        let mut trits = Vec::new();
        let mut n = cp;
        while n > 0 {
            trits.push(match n % 3 {
                0 => Trit::Zero,
                1 => Trit::One,
                _ => Trit::Two,
            });
            n /= 3;
        }
        trits.reverse();
        TritSeq(trits.into_boxed_slice())
    }

    pub fn to_hash_string(&self) -> String {
        self.0.iter().map(|t| match t {
            Trit::Zero => '0',
            Trit::One  => '1',
            Trit::Two  => '2',
        }).collect()
    }

    fn to_le_trits(&self) -> Vec<Trit> {
        let mut v = self.0.to_vec();
        v.reverse();
        v
    }

    fn from_le_trits(le: &[Trit]) -> Self {
        let mut trimmed: Vec<Trit> = le.iter()
            .rev()
            .skip_while(|&&t| t == Trit::Zero)
            .copied()
            .collect();
        if trimmed.is_empty() {
            trimmed.push(Trit::Zero);
        } else {
            trimmed.reverse();
        }
        TritSeq(trimmed.into_boxed_slice())
    }

    /// Pure ternary addition via 27-entry lookup table.
    /// No `as u8` casts, no binary arithmetic.
    pub fn add(a: &TritSeq, b: &TritSeq) -> TritSeq {
        use Trit::*;

        fn trit_full_add(a: Trit, b: Trit, c: Trit) -> (Trit, Trit) {
            match (a, b, c) {
                (Zero, Zero, Zero) => (Zero, Zero),
                (One,  Zero, Zero) => (One,  Zero),
                (Zero, One,  Zero) => (One,  Zero),
                (Zero, Zero, One)  => (One,  Zero),
                (Two,  Zero, Zero) => (Two,  Zero),
                (Zero, Two,  Zero) => (Two,  Zero),
                (Zero, Zero, Two)  => (Two,  Zero),
                (One,  One,  Zero) => (Two,  Zero),
                (One,  Zero, One)  => (Two,  Zero),
                (Zero, One,  One)  => (Two,  Zero),
                (One,  One,  One)  => (Zero, One),
                (Two,  One,  Zero) => (Zero, One),
                (Two,  Zero, One)  => (Zero, One),
                (One,  Two,  Zero) => (Zero, One),
                (Zero, Two,  One)  => (Zero, One),
                (One,  Zero, Two)  => (Zero, One),
                (Zero, One,  Two)  => (Zero, One),
                (Two,  Two,  Zero) => (One,  One),
                (Two,  Zero, Two)  => (One,  One),
                (Zero, Two,  Two)  => (One,  One),
                (Two,  One,  One)  => (One,  One),
                (One,  Two,  One)  => (One,  One),
                (One,  One,  Two)  => (One,  One),
                (Two,  Two,  One)  => (Two,  One),
                (Two,  One,  Two)  => (Two,  One),
                (One,  Two,  Two)  => (Two,  One),
                (Two,  Two,  Two)  => (Zero, Two),
            }
        }

        let a_le = a.to_le_trits();
        let b_le = b.to_le_trits();
        let max_len = a_le.len().max(b_le.len());
        let mut result_le = Vec::with_capacity(max_len + 1);
        let mut carry = Zero;
        for i in 0..max_len {
            let ai = a_le.get(i).copied().unwrap_or(Zero);
            let bi = b_le.get(i).copied().unwrap_or(Zero);
            let (digit, new_carry) = trit_full_add(ai, bi, carry);
            result_le.push(digit);
            carry = new_carry;
        }
        if carry != Zero {
            result_le.push(carry);
        }
        TritSeq::from_le_trits(&result_le)
    }

    pub fn to_base27_digits(&self) -> Vec<u8> {
        let len = self.0.len();
        let padded_len = if len % 3 == 0 { len } else { len + (3 - len % 3) };
        let mut padded = vec![Trit::Zero; padded_len - len];
        padded.extend_from_slice(&self.0);
        let mut digits = Vec::new();
        for chunk in padded.chunks(3) {
            let h = chunk[0] as u8;
            let m = chunk[1] as u8;
            let l = chunk[2] as u8;
            digits.push(h * 9 + m * 3 + l);
        }
        digits
    }
}

impl Ord for TritSeq {
    fn cmp(&self, other: &Self) -> Ordering {
        match (self.0.len(), other.0.len()) {
            (1, 1) if self.0[0] == Trit::Zero && other.0[0] == Trit::Zero =>
                Ordering::Equal,
            (1, _) if self.0[0] == Trit::Zero => Ordering::Less,
            (_, 1) if other.0[0] == Trit::Zero => Ordering::Greater,
            (a, b) if a != b => a.cmp(&b),
            _ => self.0.iter().cmp(other.0.iter()),
        }
    }
}
impl PartialOrd for TritSeq {
    fn partial_cmp(&self, other: &Self) -> Option<Ordering> { Some(self.cmp(other)) }
}
```

## 6. Immutable Persistent AVL Tree with `Rc`

```rust
use std::rc::Rc;
use std::cmp::Ordering;

#[derive(Clone, PartialEq, Eq)]
pub enum Tree<K: Ord + Clone, V: Clone> {
    Nil,
    Node { key: K, value: V, left: Rc<Tree<K, V>>, right: Rc<Tree<K, V>>, height: usize },
}

impl<K: Ord + Clone, V: Clone> Tree<K, V> {
    fn height(&self) -> usize {
        match self { Tree::Nil => 0, Tree::Node { height, .. } => *height }
    }
    fn new_node(key: K, value: V, left: Rc<Self>, right: Rc<Self>) -> Rc<Self> {
        let h = 1 + left.height().max(right.height());
        Rc::new(Tree::Node { key, value, left, right, height: h })
    }
    pub fn find(self: &Rc<Self>, key: &K) -> Option<&V> {
        match self.as_ref() {
            Tree::Nil => None,
            Tree::Node { key: k, value: v, left, right, .. } => match key.cmp(k) {
                Ordering::Less    => left.find(key),
                Ordering::Greater => right.find(key),
                Ordering::Equal   => Some(v),
            },
        }
    }
    pub fn insert(self: &Rc<Self>, key: K, value: V) -> Rc<Self> {
        match self.as_ref() {
            Tree::Nil => Tree::new_node(key, value, Rc::new(Tree::Nil), Rc::new(Tree::Nil)),
            Tree::Node { key: k, value: v, left, right, .. } => match key.cmp(k) {
                Ordering::Less => {
                    let new_left = left.insert(key, value);
                    Self::rebalance(k.clone(), v.clone(), new_left, Rc::clone(right))
                }
                Ordering::Greater => {
                    let new_right = right.insert(key, value);
                    Self::rebalance(k.clone(), v.clone(), Rc::clone(left), new_right)
                }
                Ordering::Equal => Tree::new_node(key, value, Rc::clone(left), Rc::clone(right)),
            },
        }
    }
    fn rebalance(key: K, value: V, left: Rc<Self>, right: Rc<Self>) -> Rc<Self> {
        let node = Tree::new_node(key, value, left, right);
        let bf = node.balance_factor();
        if bf == 2 {
            if node.left.balance_factor() >= 0 { Self::rotate_right(&node) }
            else {
                let lr = Self::rotate_left(&node.left);
                Self::rotate_right(&Tree::new_node(node.key.clone(), node.value.clone(), lr, Rc::clone(&node.right)))
            }
        } else if bf == -2 {
            if node.right.balance_factor() <= 0 { Self::rotate_left(&node) }
            else {
                let rr = Self::rotate_right(&node.right);
                Self::rotate_left(&Tree::new_node(node.key.clone(), node.value.clone(), Rc::clone(&node.left), rr))
            }
        } else { node }
    }
    fn balance_factor(&self) -> i8 {
        match self { Tree::Nil => 0, Tree::Node { left, right, .. } => left.height() as i8 - right.height() as i8 }
    }
    fn rotate_left(node: &Rc<Self>) -> Rc<Self> {
        if let Tree::Node { key, value, left, right, .. } = node.as_ref() {
            if let Tree::Node { key: rk, value: rv, left: rl, right: rr, .. } = right.as_ref() {
                let nl = Tree::new_node(key.clone(), value.clone(), Rc::clone(left), Rc::clone(rl));
                return Tree::new_node(rk.clone(), rv.clone(), nl, Rc::clone(rr));
            }
        }
        Rc::clone(node)
    }
    fn rotate_right(node: &Rc<Self>) -> Rc<Self> {
        if let Tree::Node { key, value, left, right, .. } = node.as_ref() {
            if let Tree::Node { key: lk, value: lv, left: ll, right: lr, .. } = left.as_ref() {
                let nr = Tree::new_node(key.clone(), value.clone(), Rc::clone(lr), Rc::clone(right));
                return Tree::new_node(lk.clone(), lv.clone(), Rc::clone(ll), nr);
            }
        }
        Rc::clone(node)
    }
}
```

```rust
fn is_isomorphic_immutable(s: &str, t: &str) -> bool {

    let t_seqs: Vec<IndexedTritSeq> = t.chars().map(IndexedTritSeq::from_char).collect();

        map: Rc<Tree<IndexedTritSeq, IndexedTritSeq>>,
        used: Rc<Tree<IndexedTritSeq, IndexedTritSeq>>,

        let sc = &s_seqs[idx]; let tc = &t_seqs[idx];

            Some(mapped) if mapped != tc => false,

                recurse(idx + 1, s_seqs, t_seqs, new_map, new_used)
            }
        }
    }
    let empty = Rc::new(Tree::Nil);
    recurse(0, &s_seqs, &t_seqs, Rc::clone(&empty), Rc::clone(&empty))
```
